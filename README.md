# ESEC Student Leave Portal

A Full Stack Web Application for managing student leave requests.

## Features
- Student Login
- Apply Leave
- Leave Calendar
- Advisor Approval System
- Approve / Reject Leave
- Leave Records
- Notification System

## Technologies Used
Frontend
- React
- Vite
- TypeScript
- Tailwind CSS
Backend
- Node.js
- Express.js
Database
- MongoDB

## Project Structure
src/
components/
pages/
contexts/
hooks/


## How to Run the Project

Install dependencies
npm install
Start development server
npm run dev
Open browser ➜  Local:   http://localhost:8080/
             ➜  Network: http://10.237.114.141:8080/
             ➜  Network: http://172.28.0.1:8080/
